Date discipline

The Inv date column shows the actual invoice date (delivery date preferred, else the date extracted from the PDF). The Imported column shows when the email landed. They're now distinct — a Jan invoice imported in March no longer masquerades as a March row.

Rows with no extracted date show — in Inv date. That means OCR hasn't run on the PDF yet. u61 cron extracts hourly at :20.
